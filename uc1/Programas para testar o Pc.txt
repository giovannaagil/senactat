Programas para testes do PC

- Passmark
Descrição do app:
 O PassMark Performance Test é uma ferramenta de benchmark que tem como objetivo avaliar o desempenho geral do computador. Além de determinar com precisão a capacidade do hardware usado, o programa serve como uma ferramenta para comparar máquinas semelhantes e determinar qual delas é a melhor.
Link para download:
https://www.passmark.com/products/performancetest/download.php

- Aida 64 Extreme
Descrição do app:
 O AIDA64 é um vasculhador de informações de hardware pra Windows e Android. O programa mostra detalhes sobre os componentes de sistema, avalia o desempenho e aponta quais são os itens instalados em um computador. ... O AIDA64 coloca nas mãos do usuário muitas informações úteis sobre o hardware que ele usa.
Link para download:
https://www.aida64.com/downloads

- Diag (Windows)
Descrição do app: 
O Windows Memory Diagnostic é um software leve e gratuito que faz testes e procura na memória RAM todos os erros ou falhas relacionadas ao hardware. ... O principal objetivo do programa é indicar ao usuário se os problemas do windows, como travar, congelar a imagem e demorar na hora de executar, são problemas de hardware

- HWMONITOR
Descrição do app: 
O CPUID HWMonitor é um aplicativo gratuito para PCs com Windows que permite verificar o desempenho do computador. O programa exibe informações como a voltagem e temperatura da CPU, utilização da memória RAM e do processador da placa de vídeo, além de dados sobre a bateria, no caso de notebooks.
Link para download:
https://www.cpuid.com/softwares/hwmonitor.html
